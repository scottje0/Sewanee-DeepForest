## Data
This folder contains all data we used to train our model. To successfully run the Scripts templates with these images, structure your folders in Jupyter Notebook identically to the structure found here, but with all of these subfolders contained within one folder titled 'df info'.

Most files found here are already formatted correctly to be uploaded directly, but some will need to be converted. Instructions for conversion are found within each applicable folder and in Scripts/UploadingData.
