# Sewanee-DeepForest
A locally trained DeepForest model for use on the Sewanee Domain that includes object detection of tree crowns and CropModel species detection capabilities.

## Loading the Model
DeepForest runs on Python, so the first step in loading the program is installing Anaconda Navigator and creating an environment for DeepForest to run in. From that environment, launch a terminal and type `jupyter lab` in the prompt. This will launch Jupyter Notebook, which is where DeepForest actually runs.
